# Yanrix

**Continuous STRIDE threat modeling for every pull request.**

Yanrix analyzes pull request diffs and produces structured STRIDE threat models that accumulate across your codebase over time. No diagrams. No meetings. No new workflow.

---

## Status

Yanrix is currently in private beta.

**[Join the waitlist at yanrix.dev](https://yanrix.dev)**

---

## How It Works

Yanrix runs as a GitHub Action on every pull request. It applies
STRIDE analysis to the diff and posts a structured threat model
directly in the PR, alongside your existing review workflow.

The threat model accumulates across the codebase. Each PR builds on
the last. The result is a living threat model that reflects the
current state of your code, not a diagram from last quarter's design meeting.
